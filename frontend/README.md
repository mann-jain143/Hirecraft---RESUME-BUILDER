# HireCraftt — Frontend Client

This is the React + Vite frontend client application for the **HireCraftt** AI Career Operating System.

## 🚀 Features

*   **Premium UI/UX**: Custom responsive sidebar layouts, glassmorphism designs, dark/light theme switching, and smooth framer-motion transitions.
*   **ATS Scoring**: Real-time evaluation of resume sections with grading widgets.
*   **Interview Prep**: Speech recognition-based real-time interview training.
*   **Hosted Portfolios**: Custom sub-routing preview and dynamic themes for personal websites.

## ⚙️ Development

Start the frontend development server:
```bash
npm run dev
```

Build production bundle:
```bash
npm run build
```
